# 👜 Mini Bag Store – Full Stack Project

A simplified full-stack e-commerce web application for a **Mini Bag Store**, built using **React, Node.js, Express, and MongoDB**.  
This project simulates a real-world shopping experience with product listing, cart management, checkout, and order confirmation.

---

## 🚀 Features

- 🏠 **Home / Shop Page** – Displays products fetched from backend
- 📄 **Product Details Page** – Full product information with Add to Cart
- 🛒 **Cart Page**
  - View added items
  - Remove items
  - View total price
- 🧾 **Checkout Page**
  - Name
  - Email
  - Address
- ✅ **Order Success Page** – Confirmation after successful checkout
- 📱 **Fully Responsive UI** (Mobile & Desktop)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Bootstrap & React-Bootstrap
- Fetch API

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
